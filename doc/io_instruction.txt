IO Instructions
===============

IO Instructions normally perform operations on the first Accumulator (AC0)

Port 0x00:	CPU Internal (CPU)
Code:
EI	0x0:	Enable Interrupt
DI	0x1:	Disable Interrupt
CAF	0x2:	Clear All Flags
SKIE	0x3:	If Interrupt is enabled, skip next instruction
SKGT	0x5:	If GT == 1, skip next instruction
SKDV	0x4:	If STAT == 2 (Device), skip next instruction
SKSC	0x6:	if STAT == 3 (System Call), skip next instruction
SKUI	0x7:	if STAT == 4 (Usermode Interrupt), skip next instruction
SKTP	0x8:	if STAT == 5 (Trap), skip next instruction
SKNP	0x9:	if STAT == 6 (No Permission), skip next instruction
SST	0xA:	Set STATUS
GST	0xB:	Get STATUS
RST	0xC:	Restore STATUS
	0xD:
	0xE:
HALT	0xF:	Halt

Port 0x01:	Main Console (CON)
Code:
EII	0x0:	Enable Input Interrupt
EOI	0x1:	Enable Output Interrupt
DII	0x2:	Disable Input Interrupt
DOI	0x3:	Disable Output Interrupt
SIF	0x4:	Set Input Flag
SOF	0x5:	Set Output Flag
CIF	0x6:	Clear Input Flag
CIA	0x7:	Clear Input Flag and AC0
COF	0x8:	Clear Output Flag
IFSK	0x9:	If Input Flag is On, skip next instruction
OFSK	0xA:	If Output Flag is set, skip next instruction
IOSK	0xB:	If Input or Output Flag is set, skip next instruction
RA	0xC:	Read Input (OR) into AC0 (Lower 8 Bit)
WA	0xD:	Write content of AC0 (Lower 8 Bit)
CRA	0xE:	Clear Input Flag and AC0, then Read Input into AC0
CWA	0xF:	Clear Output Flag and Write content of AC0 (Lower 8 Bit)

Port 0x02:	Paper Tape Reader/Puncher (PT)
Code:
EI	0x0:	Enable Interrupt
DI	0x1:	Disable Interrupt
RFSK	0x2:	If Read Flag is set, skip next instruction
PFSK	0x3:	If Punch Flag is set, skip next instruction
CRF	0x4:	Clear Read Flag
CPF	0x5:	Clear Punch Flag
RPA	0x6:	Read into AC0 (Lower 8 Bit, OR)
PPA	0x7:	Punch AC0 (Lower 8 Bit)
CRR	0x8:	Clear Read Flag, and Read into AC0 (Lower 8 Bit, OR)
CPP	0x9:	Clear Punch Flag, Punch AC0 (Lower 8 Bit)
	0xA:
	0xB:
	0xC:
	0xD:
	0xE:
	0xF:

Note:
	Interrupts 10 times per second
Code:
EI	0x0:	Enable Interrupt
DI	0x1:	Disable Interrupt
CSK	0x2:	If is on Clock Flag is set, skip next instruction
CCF	0x3:	Clear Clock Flag

Port 0x04:	Time Sharing Controller (TSC)
Note:
	Default is to interrupt 100 times per second
Code:
EI	0x0:	Enable Interrupt
DI	0x1:	Disable Interrupt
PTCS	0x2:	If is on the point to do context switch, skip next instruction
CLPF	0x3:	Clear Context Switch Flag
SETC	0x4:	Set Cycle Time of Context Switch, in micro-second per cycle
GETC	0x5:	Get Cycle Time of Context Switch

	0x7:
	0x8:

Port 0x05:	Memory Management Controller (MMC)
Code:
	0x0:	Set Code Field to Lower 8 Bit
	0x1:	Set Stack Field to Lower 8 Bit
	0x2:	Set Data Field to Lower 8 Bit
	0x3:	Set Indirect Field to Lower 8 Bit
	0x4:	Set Code Field (High) and Stack Field (Low)
	0x5:	Set Data Field (High) and Indirect Field (Low)
	0x6:	Get Code Field
	0x7:	Get Stack Field
	0x8:	Get Data Field
	0x9:	Get Indirect Field
	0xA:	Get Code Field (High) and Stack Field (Low)
	0xB:	Set Data Field (High) and Indirect Field (Low)
	0xC:
	0xD:
	0xE:	Enable Fallthrough Field
	0xF:	Disable Fallthrough Field
