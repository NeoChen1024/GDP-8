IO Instructions
===============

Device:	CPU	(0x00)
	EI:	Enable Interrupt
	DI:	Disable
	ECLK:	Enable Clock Interrupt
	DCLK:	Disable
	PSR:	Read panel switches (to Register)
	DPY:	Display Custom data on panel display
	ABRT:	Reboot, Clear memory, registers, and flags
	GIV:	Get Interrupt Vector
Device: CONSOLE	(0x01)
	EI:	Enable Interrupt
	DI:	Disable Interrupt
	OUT:	Write one byte
	IN:	Read one byte
	CLR:	Clear screen
	BEL:	Send Bell
	RST:	Reset console
